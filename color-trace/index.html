👆 Tap anywhere — watch for color in safe areas

How to read this

In normal state, the safe areas show RED (html background) or ORANGE (body covers html). During the 2-second transition, if the color changes, a VT pseudo-element is painting over the page.

Expected findings

If the safe area turns MAGENTA → ::view-transition is the source. If it turns BLUE/PINK → the old/new root snapshots are the source. If it stays RED/ORANGE → it's coming from html/body, not VT.

If WHITE appears

If none of the colors above match and you see WHITE, then the color source is outside our CSS — Safari's default compositing or system theme.

Test environment

Safari iOS 18.2+ on notched iPhone. Watch carefully during the 2-second animation.

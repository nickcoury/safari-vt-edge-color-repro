Variant 7
Types-Scoped Transitions. Uses startViewTransition({ update, types: ['card-expand'] }). The :active-view-transition-type(card-expand) pseudo-class scopes the root animation suppression to only this specific transition type. Other transitions (if any) would not be affected.
// JS: document.startViewTransition({ update: () => updateCardState(), types: ['card-expand'] }); // CSS: html:active-view-transition-type(card-expand) ::view-transition-old(root), html:active-view-transition-type(card-expand) ::view-transition-new(root) { animation: none; }
View Transition Card

This variant uses the types parameter of startViewTransition() to scope VT CSS. The root animation is disabled only when the active transition type is card-expand.

Types allow different transitions (expand, collapse, navigate, etc.) to have different CSS overrides — useful if you want root animation on navigation but not on card expansion.

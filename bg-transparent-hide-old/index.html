Variant 14
Combo: Transparent BG + Hide Root Old. Combines variant 3 and variant 9 — the old root snapshot is both hidden (display: none) and has transparent background, while the new root snapshot also gets transparent background in case Safari still composites it.
::view-transition-old(root) { background: transparent; display: none; } ::view-transition-new(root) { background: transparent; }
View Transition Card

The old root snapshot is hidden from rendering and has transparent background. The new root snapshot — which replaces it immediately — also gets transparent background as a safety net.

If the new root snapshot is the one causing the white flash (Safari might snapshot the new page state and composite it over the old), this targets both the old and new snapshots.

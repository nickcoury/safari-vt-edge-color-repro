Background: Transparent on ::view-transition Overlay. The top-level ::view-transition pseudo-element (the root of the entire VT tree) gets background: transparent. This is the outermost overlay that sits on top of the page during the transition.
::view-transition { background: transparent; }
View Transition Card

The ::view-transition pseudo-element is the top-level overlay that contains all VT groups. It's a fixed-position element covering the entire viewport. If Safari derives bar colors from this outermost layer's background, setting it to transparent might help.

This is the highest-level element in the VT pseudo-element tree — above all groups and image pairs.

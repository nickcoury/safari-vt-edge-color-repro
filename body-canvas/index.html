Body = Canvas. Uses the CSS system color keyword Canvas, which represents the system's native background color. On iOS, this is the same color used for native UI backgrounds. If Safari samples body's background for safe areas, this should match the system bars natively.
body { background: Canvas; }
View Transition Card

Canvas is a CSS system color that resolves to the operating system's default background color. On iOS in light mode, it's typically white or near-white. In dark mode, it's dark.

If Safari samples body's background for the safe area color, using Canvas should make the safe areas blend with the system's native appearance rather than flashing a mismatched color.

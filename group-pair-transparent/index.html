Variant 32
Group + Image-Pair Transparent. Sets background: transparent on both ::view-transition-group(root) AND ::view-transition-image-pair(root) together, plus old/new. Previous variants tested these individually but not the specific combination of group + image-pair.
::view-transition-group(root), ::view-transition-image-pair(root), ::view-transition-old(root), ::view-transition-new(root) { background: transparent !important; }
View Transition Card

The ::view-transition-image-pair(root) is the container that holds both the old and new snapshot images. It sits between the group and the individual old/new pseudo-elements.

If this layer has an opaque background (even inherited), it would paint over the safe areas. Making it explicitly transparent closes that gap.
